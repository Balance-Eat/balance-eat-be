## 요구사항
- 사용자는 본인이 등록한 기기의 알림 활성/비활성화 상태를 변경할 수 있어야 한다.


## API Specification

## 엔드포인트
(PATCH) /v1/notification-devices/{deviceId}/activation

## 요청 명세
### 요청 헤더 (Request Header)
- `X-USER-ID`: {userId} (Long, 필수)

### 요청 바디 (Request Body)
```json
{
  "isActive": true | false // 필수, 알림 수신 허용 여부
}
```

## 응답 명세
### 응답 바디 (Response Body)
```json
{
    "status": "SUCCESS",
    "message": "성공",
    "data": {
        "id": 12345,
        "isActive": true
    },
    "serverDatetime": "2025-09-28T19:12:07.
}
