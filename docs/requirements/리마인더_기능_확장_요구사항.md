## 추가 요구사항
- isActive 값을 통해 활성 비활성화를 할 수 있다.
- dayOfWeeks 값을 통해 알림을 받을 요일을 설정할 수 있다.
  - requestBody: [1 ~ 7] (1: 월요일, 2: 화요일, ..., 7: 일요일)
  - entity: [DayOfWeek.MONDAY, DayOfWeek.TUESDAY, ..., DayOfWeek.SUNDAY] DayOfWeek enum 사용
  - db column: json integer array (Converter 를 통해 enum <-> db column 매핑)
- sendDatetime -> sendTime 으로 필드명 변경 (HH:mm:ss LocalTime 형식)
  - sendTime의 ss는 0으로 고정
