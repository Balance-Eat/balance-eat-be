## 기술 스택
spring scheduler, spring jdbc 

버전은 현재 스펙에 맞게 최대한 유사하게

## 요구사항
- jobParameter
  - targetDate: yyyyMMdd
- cleanup tasklet
  - diet_stats 테이블에서 target_date = jobParameter.targetDate 인 데이터 삭제
- chunk size: 1000
- reader
  - 모든 유저 대상 통계 정보 조회 (DietStats.kt 모든 필드)
    - userId
    - targetDate
    - totalKcal: 해당 일자에 섭취한 총 칼로리
    - totalCarb: 해당 일자에 섭취한 총 탄수화물
    - totalProtein: 해당 일자에 섭취한 총 단백질
    - totalFat: 해당 일자에 섭취한 총 지방
- writer:
-  집계된 DietStats 클래스 배치 인서트
