(PUT) /v1/diets/{dietId}

## 요청 명세
### 요청 헤더 (Request Header)
- `X-USER-ID`: {userId} (Long, 필수)

```json
  "mealType": "BREAKFAST",
  "consumedAt": "2024-01-15T08:30:00",

      "intake": 2

}
```
