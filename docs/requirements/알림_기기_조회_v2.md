## 요구사항
- 사용자는 자신의 식별자와 디바이스의 agentId를 통해 알림 디바이스 정보를 조회할 수 있다.

## API Specification

## 엔드포인트
(GET) /v1/notification-devices/current

## 요청 명세
### 요청 헤더 (Request Header)
- `X-USER-ID`: {userId} (Long, 필수)
- `X-Device-Agent-Id`: {agentId} (String, 필수)
